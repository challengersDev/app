# Файл конфигурации фильтров.
#
# Фильтры бывают трех типов: главный, временной и обычный.
# У каждого должно быть имя, русскоязычное название и хотя бы 1 значение
#
# Главный только один - значения выводятся после слова mainFilter
# и завершаются /mainFilter. Первое значение - имя пункта "ВСЕ":
# упражнения не фильтруются.
#
# Временной - имя начинается с timeFilter[Name], значения считываются
# до /timeFilter[Name] (например, timeFilter23 ... /timeFilter23)
# Значения 5 10 15 преобразуются в "ДО 5 МИНУТ", "ОТ 5 ДО 10 МИНУТ",
# "ОТ 10 ДО 15 МИНУТ", "ОТ 15 МИНУТ".
#
# Обычный - имя начинается с filter[Name], значения считываются 
# до /filter[Name]. Первое значение - название фильтра.

# Для фильтрации используется имя фильтра [Name], 
# Которое должно соответствовать имени поля для упражнения.

# Файл конфигурации отображения упражнения
#
# Упражнение имеет название, картинку, описание,
# а также шаги выполнения и ошибки.
#
# В данном файле добавляются остальные аттрибуты упражнения,
# указывается, какие из них будут отображаться и на какой странице.
# Имя аттрибута должно совпадать с [Name] в filter[Name] (filter.txt),
# чтобы работала фильтрация по данному аттрибуту.
# ! В таком же порядке добавляются данный из базы данных !
#
# Первые три строки отвечают за вывод на экран на страничке каталога
# Вторые три строки - добавляются к первым при выводе на страничке описания
# Остальные строки указывают другие типы, которые не будут отображаться на экране,
# Но по ним также может производиться фильтрация.
